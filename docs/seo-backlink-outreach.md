# Backlink outreach (months 4–6)

Low-spam tactics for the Teacher portfolio sites.

## Targets

- Dev.to / Hashnode: republish pillar blog excerpts with canonical links to `/blog/{slug}`.
- GitHub awesome lists: `awesome-interview-questions`, `awesome-python`, `awesome-react`, `awesome-nodejs`.
- Bootcamp resource pages: offer free prep links in exchange for syllabus mentions.
- Reddit r/learnpython, r/reactjs, r/node (follow sub rules; link only when genuinely helpful).

## Anchor text examples

- "free Python interview challenges"
- "React hooks flashcards"
- "Node.js system design walkthrough"

## Cross-linking (owned)

Footer on each site links to the other two Teacher sites (`PORTFOLIO_SITES` in `seo-content.ts`). No reciprocal link schemes with third parties.

## Tracking

Record placements in a spreadsheet: URL, anchor, date, follow/nofollow, estimated traffic.
